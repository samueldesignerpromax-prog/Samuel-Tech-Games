## 📋 Sobre o Projeto

O **Samuel Tech Games** é um portal de jogos indie onde os usuários podem:
- 🎮 Jogar 7 jogos diferentes online
- ⭐ Avaliar os jogos de 1 a 5 estrelas
- 💬 Deixar comentários que ficam salvos na nuvem
- 📱 Acessar pelo celular, PC ou tablet (todos os comentários sincronizados)

## 🛠️ Tecnologias Utilizadas

### Frontend
- HTML5
- CSS3 (com animações e responsividade)
- JavaScript (Vanilla)
- Canvas para efeito Matrix

### Backend
- Node.js
- Express.js
- Sistema de API própria para comentários

### Hospedagem
- Render.com (backend + frontend)
